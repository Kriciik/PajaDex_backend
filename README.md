## Pokemon Backend
Backend for PajaDex mainly used as a data provider for frontend part of the application through endpoints made using <b>Node.js</b> with <b>typescript</b>. <p>
PajaDex is hosted on: none

## Database
Database is running on PostgreSQL created with DrizzleORM.

### Tables
<ul>
  <li>users</li>
  <li>collection</li>
  <li>cards</li>
  <li>groups</li>
  <li>card_groups (junction table) </li>
</ul>
